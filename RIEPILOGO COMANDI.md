COMANDI RAPIDI PER AGGIORNARE REPOSITORY SU GITHUB da terminale

1.   APRI IL TERMINALE
cd /Users/donatopastrello/Desktop/donato-pastrello-website

2.   AGGIUNGI LE MODIFICHE
git add .

3.   CREA UN COMMIT CON LE MODIFICHE APPORTATE
git commit -m "Descrizione delle modifiche fatte"

4.   CARICA SU GITHUB
git push

❌ Se Il git push fallisce perché manca il collegamento upstream, digitare su terminale:
git push -u origin main
